# Описание проекта

"Boss Rush" - это динамичный 2D-платформер с элементами экшена, где игрок сражается с умным боссом на многоуровневой арене. Игра сочетает в себе точное управление персонажем, тактическое перемещение по платформам и стратегическое противостояние с продвинутым ИИ противника. Проект включает полноценную систему аккаунтов, магазин улучшений и прогрессивную систему развития персонажа.

## Технические требования

- **Python**: версия 3.8 или выше
- **Оперативная память**: не менее 2 ГБ
- **Свободное место на диске**: не менее 50 МБ
- **Операционная система**: Windows 7/8/10/11, Linux, macOS

## Инструкция по установке и запуску

### Шаг 1: Установка Python
1. Скачайте Python с официального сайта python.org
2. Запустите установщик
3. **ВАЖНО**: Поставьте галочку "Add Python to PATH"
4. Нажмите "Install Now"
5. Дождитесь завершения установки

### Шаг 2: Проверка установки Python
1. Откройте командную строку (Win + R, введите cmd, нажмите Enter)
2. Введите команду: `python --version`
3. Если отображается версия Python (например, "Python 3.8.0"), установка прошла успешно

### Шаг 3: Подготовка файлов проекта
1. Распакуйте архив с проектом в любую папку
2. Убедитесь, что в папке проекта есть следующие файлы:
   - `Python.py` (главный файл приложения)
   - `game_engine.py` (игровой движок)
   - `database.py` (работа с базой данных)
   - `sound_manager.py` (управление звуками)
   - `constants.py` (конфигурационные константы)
   - `requirements.txt` (список зависимостей)
   - Файлы интерфейсов: `start_ui.py`, `register_ui.py`, `login_ui.py`, `main_menu_ui.py`, `profile_ui.py`, `shop_ui.py`
   - Папка `sounds/` с файлом `bullet_shot.mp3`

### Шаг 4: Установка зависимостей
1. Откройте командную строку
2. Перейдите в папку с проектом. Например: `cd C:\Users\Имя\Desktop\BossRush`
3. Выполните команду: `pip install -r requirements.txt`
4. Дождитесь завершения установки библиотек

### Шаг 5: Запуск игры
**Способ 1:**
- Дважды щелкните по файлу `Python.py`

**Способ 2:**
- Откройте командную строку в папке проекта
- Выполните команду: `python Python.py`

## Руководство пользователя

### Регистрация и вход
- При первом запуске нажмите "Register" для создания аккаунта
- Придумайте логин (не менее 3 символов) и пароль (не менее 4 символов)
- Для последующих входов используйте кнопку "Login"

### Управление в игре
- **A** - движение влево
- **D** - движение вправо
- **Space** - прыжок
- **Enter** - выстрел
- **Escape** - пауза/меню
- **T** - тест звука (для отладки)
- Клик по кнопке "II" в правом верхнем углу - пауза

### Игровой процесс
1. После входа нажмите "Играть" в главном меню
2. Управляйте персонажем с помощью клавиш A/D
3. Прыгайте по платформам с помощью Space для тактического позиционирования
4. Стреляйте в босса клавишей Enter
5. Избегайте близкого контакта с боссом - он наносит урон при приближении
6. Используйте платформы для уклонения и поиска выгодных позиций
7. Следите за полосками здоровья своего персонажа и босса

### Система профиля
- В главном меню нажмите "Профиль" для просмотра статистики
- Можно загрузить фото профиля через кнопку "Выбрать фото"
- Отображается: никнейм, пароль, количество монет, уровень, дата регистрации
- Данные автоматически сохраняются в базе данных

### Магазин улучшений
- В главном меню нажмите "Магазин"
- Доступны три категории улучшений:
  - **Улучшения игрока**: здоровье, скорость, прыжки
  - **Улучшения оружия**: урон, скорострельность, тройной выстрел
  - **Бонусы**: регенерация, щиты, замедление времени
- Для покупки нужны монеты, которые зарабатываются за победу над боссом
- После покупки улучшение можно применить к персонажу

## Особенности реализации

### Что работает стабильно
-  Полноценная система регистрации и авторизации с валидацией данных
-  Плавное движение персонажа с реалистичной физикой прыжков и гравитацией
-  Сложная система коллизий с полом и платформами
-  Продвинутый ИИ босса с навигацией по платформам и поиском пути
-  Система здоровья, урона и боевых взаимодействий
-  Стрельба пулями с системой перезарядки и столкновений
-  Система паузы с возможностью возврата в главное меню
-  Профиль игрока с сохранением данных и аватарок
-  Магазин с системой покупок, применений улучшений и экономикой
-  Звуковая система с эффектами выстрелов
-  База данных SQLite для сохранения прогресса и статистики

### Технические особенности
- **Графический интерфейс**: PyQt6 с кастомными стилями и анимациями
- **Игровой движок**: QGraphicsScene для рендеринга игрового мира
- **База данных**: SQLite с тремя основными таблицами (players, shop_items, player_purchases)
- **Игровой цикл**: Система таймеров для плавного обновления кадров
- **Физика**: Реалистичное движение с гравитацией и буферизацией прыжков
- **AI система**: Сложные алгоритмы поиска пути и навигации по платформам
- **Архитектура**: Модульная структура с четким разделением ответственности

## Возможные проблемы и решения

### Проблема: "ModuleNotFoundError: No module named 'PyQt6'"
**Решение**: Выполните в командной строке: `pip install PyQt6`

### Проблема: Игра не запускается двойным кликом
**Решение**: Запустите через командную строку: `python Python.py`

### Проблема: Не отображаются изображения или интерфейс
**Решение**: Убедитесь, что все файлы UI находятся в той же папке, что и Python.py

### Проблема: Ошибки базы данных
**Решение**: Удалите файл `space_shooter.db` - он создастся заново при запуске

### Проблема: Звуки не воспроизводятся
**Решение**: Проверьте наличие папки `sounds/` с файлом `bullet_shot.mp3`

### Проблема: "Could not update timestamps" в консоли
**Решение**: Это предупреждения FFmpeg, не влияющие на игровой процесс. Можно игнорировать.
