# ОПИСАНИЕ ПРОЕКТА

**Название:** "Boss Rush"

**Жанр:** 2D платформер с боями против боссов

**Цель игры:** Игрок управляет персонажем, который сражается с умным боссом на многоуровневой арене с платформами.

## ТЕХНИЧЕСКОЕ ЗАДАНИЕ

### 1. ТЕХНОЛОГИИ
- **Язык программирования:** Python
- **Библиотека для окон и графики:** PyQt6
- **База данных:** SQLite
- **Графика:** QGraphicsScene для отрисовки, QPixmap для спрайтов
- **Аудио:** QMediaPlayer для звуковых эффектов

### 2. ЧТО РЕАЛИЗОВАНО

####  Полная система аутентификации
- Регистрация и вход с валидацией
- Хранение данных игроков в БД
- Система профилей с аватарками

####  Главное меню и навигация
- Современный интерфейс с кнопками
- Отображение информации об игроке
- Навигация между окнами: Игра, Магазин, Профиль

####  Магазин с системой улучшений
- 3 категории товаров: Игрок, Оружие, Бонусы
- Покупка за внутриигровую валюту
- Система применения улучшений
- Экономическая система с балансом

####  Полноценный игровой движок
- Плавное движение персонажа (A/D для движения, Space для прыжка)
- Сложная физика с гравитацией и прыжками
- Система столкновений с полом и платформами
- Анимации ходьбы и состояний

####  Продвинутый AI босса
- Умное преследование игрока
- Навигация по платформам с поиском пути
- Прыжки через препятствия
- Адаптивное поведение в зависимости от здоровья

####  Система боя
- Стрельба пулями (Enter для выстрела)
- Система урона и здоровья
- Столкновения пуль с боссом
- Визуальные полоски здоровья

####  Звуковая система
- Звуковые эффекты выстрелов
- Управление воспроизведением
- Интеграция с игровыми событиями

####  Система состояний игры
- Пауза с меню управления
- Экран победы при победе над боссом
- Экран поражения при смерти игрока
- Перезапуск игры

### 3. ХАРАКТЕРИСТИКИ СИСТЕМ

#### 3.1. Игрок
- **Здоровье:** 1000 HP (максимум)
- **Стрельба:** 
  - Выстрелы по нажатию Enter
  - Система перезарядки
  - Настраиваемый урон пуль
- **Движение:**
  - Бег влево/вправо
  - Прыжки с физикой
  - Буфер прыжка для отзывчивости

#### 3.2. Босс
- **Здоровье:** 1000 HP (максимум)
- **AI поведение:**
  - Преследование по горизонтали
  - Навигация между платформами
  - Прыжки к игроку
  - Поиск оптимального пути
- **Атака:** Приближение к игроку для нанесения урона

#### 3.3. Окружение
- **Многоуровневая арена** с платформами
- **Физические поверхности** для движения
- **Система коллизий** для реалистичного взаимодействия

### 4. ИНТЕРФЕЙС В ИГРЕ
- **Полоска здоровья игрока** с цветовой индикацией
- **Полоска здоровья босса** 
- **Имя игрока** над персонажем
- **Метка босса** с идентификацией
- **Кнопка паузы** в углу экрана
- **Меню паузы** с опциями продолжения и выхода

### 5. БАЗА ДАННЫХ
**Таблицы и данные:**
- `players` - данные игроков (логин, пароль, кредиты, уровень, фото)
- `shop_items` - товары магазина (название, описание, цена, эффекты)
- `player_purchases` - история покупок игроков

**Сохраняет:**
- Аккаунты игроков с прогрессом
- Баланс кредитов и уровни
- Купленные улучшения
- Аватары игроков

### 6. СИСТЕМА УЛУЧШЕНИЙ (МАГАЗИН)

#### Категория "Игрок":
-  Увеличение здоровья (+50 HP)
-  Ускорение бега (+20% скорости)
-  Сильный прыжок (+30% силы прыжка)

#### Категория "Оружие":
-  Усиленные пули (+50% урона)
-  Быстрая перезарядка (скорострельность x2)
-  Тройной выстрел (стреляет тремя пулями)

#### Категория "Бонусы":
-  Регенерация (5 HP/сек)
-  Защитный щит (бесплатный удар)
-  Замедление времени (замедляет босса)

### 7. УПРАВЛЕНИЕ
- **A/D** - движение влево/вправо
- **Space** - прыжок
- **Enter** - выстрел
- **Esc** - пауза/продолжение
- **Мышь** - клик по кнопкам интерфейса

### 8. АРХИТЕКТУРА ПРОЕКТА

#### Основные модули:
- **Python.py** - главный файл приложения
- **game_engine.py** - игровой движок с физикой и AI
- **database.py** - работа с SQLite базой
- **sound_manager.py** - управление звуковыми эффектами
- **constants.py** - конфигурационные константы
- **UI файлы** - интерфейсы окон (.ui)

#### Классы и компоненты:
- **StartWindow** - стартовое окно
- **EntranceWindow** - вход в аккаунт
- **RegisterWindow** - регистрация
- **MainMenuWindow** - главное меню
- **GameWindow** - игровой процесс
- **ShopWindow** - магазин улучшений
- **ProfileWindow** - профиль игрока
- **Database** - управление данными
- **SoundManager** - управление звуками
